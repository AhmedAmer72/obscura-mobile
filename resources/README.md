# Capacitor native assets

Place source images here, then run:

```bash
npm run assets:generate
```

Recommended files:

- `icon.png` — 1024×1024, Obscura mark on transparent or sage background
- `splash.png` — 2732×2732 optional; or use logo + `assets.config.json` background colors

Source SVG: `public/brand/obscura-mark.svg`

Colors (also in `assets.config.json`):

- Background: `#EEF3EA`
- Dark: `#18280E`
